# Educa-o-Financeira-Aplicada-Estudo-com-IA
s · 20 de abr. de 2026 Este projeto consiste na construção de um caderno temático de Educação Financeira, desenvolvido com o apoio de inteligência artificial, utilizando a ferramenta NotebookLM.  O objetivo foi organizar, analisar e transformar conteúdos de fontes confiáveis em um material estruturado de estudo
